# AI Skills Reference

This document describes each AI skill available in Business Builder.

## Business Architect

**Module:** `skills/business_architect.py`  
**Prompt:** `prompts/business_architect.md`

Decomposes the business idea into bounded contexts, key entities, service boundaries, data flows, and external dependencies.

---

## Revenue Model Designer

**Module:** `skills/revenue_model_designer.py`  
**Prompt:** `prompts/revenue_model_designer.md`

Evaluates the best monetisation strategy (SaaS, Freemium, Marketplace, etc.) and produces pricing tiers, revenue projections, and key metrics.

---

## Viability Analyst

**Module:** `skills/viability_analyst.py`  
**Prompt:** `prompts/viability_analyst.md`

Scores the idea's viability (0–100), maps the market opportunity, surfaces the competitive landscape, identifies top risks, and issues a Go/No-Go recommendation.

---

## Infrastructure Planner

**Module:** `skills/infrastructure_planner.py`  
**Prompt:** `prompts/infrastructure_planner.md`

Designs a cloud infrastructure blueprint including provider selection, core managed services, scaling strategy, security controls, and cost estimates.

---

## API Blueprint Generator

**Module:** `skills/api_blueprint_generator.py`  
**Prompt:** `prompts/api_blueprint_generator.md`

Produces a high-level REST/GraphQL API contract with endpoint listings, authentication strategy, error handling standards, and pagination approach.

---

## Data Model Architect

**Module:** `skills/data_model_architect.py`  
**Prompt:** `prompts/data_model_architect.md`

Generates an entity-relationship diagram, attribute tables, relationship descriptions, database engine recommendation, and indexing strategy.
